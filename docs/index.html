Features

Everything you need.
Nothing you don't.

SOVN strips away the clutter and gives you a browser that respects your time, privacy, and hardware.

⚡
Lightning Performance
Powered by Edge WebView2's Chromium engine — the same engine behind Microsoft Edge. Full V8 JavaScript, hardware-accelerated rendering, and native Windows integration for instant page loads.
🔒
Privacy by Default
Zero telemetry. Zero data collection. Your browsing history, bookmarks, and profiles are stored locally on your machine — never sent to any server. What you browse is yours alone.
🎨
Minimal Dark UI
A distraction-free interface designed from scratch. No bloated toolbars, no unwanted prompts. Every pixel is intentional — beautiful, dark, and focused on your content.
👤
Multi-Profile System
Create up to 6 independent profiles, each with their own bookmarks, history, and identity. Switch between work, personal, and more with a single click — no tab juggling required.
🔖
Smart Bookmarks
Save pages instantly with Ctrl+D. Access all your bookmarks and history from the built-in settings panel. Search across all major engines — Google, Bing, DuckDuckGo, or Brave.
🎮
Built-in Offline Game
Lost your connection? SOVN's built-in SOVN Runner game launches automatically when you go offline. A neon-themed flappy runner to keep you entertained until you're back online.
How It's Built

Engineered for
quality & speed

SOVN was built with a deliberate tech stack chosen to maximize performance, minimize dependencies, and ensure reliability on every Windows machine.

🐍
Python 3.12
Core application logic, data persistence, and API bridge
🌐
Edge WebView2 (Chromium)
Full Chromium rendering engine — V8 JS, CSS Grid, WebAssembly
🪟
pywebview
Native Windows window integration with Python↔JS bridge
📦
PyInstaller + Inno Setup
Compiled to a self-contained EXE with a professional installer
Quality Guarantees
Hardware-Accelerated Rendering
WebView2 leverages your GPU for compositing and animations — smooth 60fps scrolling and rendering on any modern Windows PC.
Persistent File-Based Storage
Bookmarks, history, and profiles are stored as JSON files in %USERPROFILE%\.sovn\ — reliable across sessions, restarts, and updates.
Lightweight & Fast Startup
SOVN launches in under 2 seconds on standard hardware. The bundled app is under 15MB — no Electron, no Node.js, no bloat.
Chromium Security Model
WebView2 receives automatic security updates from Microsoft Edge. You always browse with an up-to-date, sandboxed Chromium engine.
